Key Concepts

Least Cost Routing (LCR)

Kamailio evaluates registered carrier rate tables on every outbound call and routes to the cheapest carrier for the destination prefix. Daily automated CSV import keeps rates current. Automatic failover to next cheapest if carrier is unreachable.

Real-time Billing (A2Billing)

Every call triggers an AGI script in Asterisk that queries A2Billing before connecting. Insufficient balance = immediate rejection. Sub-second decision over the isolated Data network.

Isolated vSwitch Air Gap

Voice, Data, and Sync networks have no physical uplink — not a firewall rule, not an ACL. No wire connects them to the internet. MariaDB nodes are physically unreachable from outside regardless of any software configuration.

MTU 9000 Jumbo Frames

Internal networks run at MTU 9000 — 6× larger packets than standard Ethernet. Fewer interrupts per MB transferred means measurable CPU reduction on MariaDB replication, A2Billing CDR writes, and SIP/RTP inter-service flows.
